# Backend API

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend API design and implementation.

## When to use this skill:

- Creating new API endpoints or routes
- Designing RESTful resource structures
- Implementing or modifying HTTP request/response handlers
- Setting up query parameters, filters, and pagination
- Configuring API versioning strategies
- Adding rate limiting or response headers to API responses
- Working with route definitions and endpoint organization
- Implementing API middleware or interceptors
- Designing nested resource endpoints
- Managing API consistency across multiple endpoints
