# Backend Migrations

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend database migrations.

## When to use this skill:

- Creating new database migration scripts
- Modifying database schema with migrations
- Adding or removing database columns or tables
- Implementing database version control
- Writing rollback procedures for migrations
- Managing data transformations during schema changes
- Coordinating migrations across multiple environments
- Organizing migration file structures
- Testing migration reversibility
- Documenting migration changes
