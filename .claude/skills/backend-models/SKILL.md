# Backend Models

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle backend data models and entities.

## When to use this skill:

- Defining new data model classes or entities
- Creating database entity structures
- Establishing relationships between models (one-to-many, many-to-many, etc.)
- Adding validation logic to models
- Implementing computed properties or derived fields
- Managing model inheritance and composition
- Creating model interfaces or type definitions
- Implementing model serialization/deserialization
- Adding lifecycle hooks to models
- Organizing model files and structures
