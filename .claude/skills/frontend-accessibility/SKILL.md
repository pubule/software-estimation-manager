# Frontend Accessibility

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend accessibility and inclusive design.

## When to use this skill:

- Creating accessible UI components
- Adding semantic HTML elements
- Implementing keyboard navigation
- Setting up focus management and tab order
- Adding ARIA labels and attributes
- Ensuring proper color contrast ratios
- Building accessible forms with proper labels
- Implementing screen reader support
- Adding skip links and navigation landmarks
- Testing and validating accessibility compliance
- Creating accessible data tables and lists
