# Frontend Components

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend component design and development.

## When to use this skill:

- Creating new React or UI components
- Designing component hierarchies and composition
- Defining component props interfaces
- Implementing single-responsibility components
- Building reusable component patterns
- Managing component internal state
- Creating component lifecycle logic
- Establishing clear component APIs
- Documenting component usage and examples
- Refactoring components for reusability
- Managing prop drilling and data flow
- Building component composition patterns
