# Frontend CSS

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend styling and CSS.

## When to use this skill:

- Writing component-scoped styles
- Creating utility-first CSS classes
- Implementing consistent design systems
- Setting up CSS naming conventions
- Building responsive styles
- Creating reusable CSS patterns
- Managing CSS specificity
- Implementing CSS custom properties (variables)
- Setting up CSS modules or CSS-in-JS
- Organizing stylesheet structures
- Creating component theme systems
- Optimizing CSS for performance
