# Global Coding Style

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle global coding style and consistency.

## When to use this skill:

- Writing any new code in the project
- Establishing consistent naming conventions
- Choosing meaningful variable and function names
- Breaking down large functions into smaller focused ones
- Removing code duplication and following DRY principles
- Maintaining consistent indentation and formatting
- Deleting unused code and dead imports
- Organizing logical code structure
- Applying backward compatibility only when required
- Following team coding conventions
- Writing self-documenting code
- Refactoring for clarity and maintainability
