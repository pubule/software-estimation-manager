# Global Commenting

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle comments and code documentation.

## When to use this skill:

- Writing comments for complex logic
- Explaining non-obvious implementation decisions
- Documenting function and class purposes
- Adding JSDoc/TSDoc documentation
- Clarifying business logic and edge cases
- Explaining algorithm implementations
- Documenting public API functions
- Adding comments for architectural patterns
- Explaining workarounds and temporary solutions
- Documenting parameter meanings and return values
- Commenting on performance considerations
- Explaining integration points and dependencies
