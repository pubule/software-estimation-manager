# Global Validation

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle data validation across the application.

## When to use this skill:

- Validating form inputs and user data
- Implementing input sanitization
- Checking function parameter types and values
- Validating API request payloads
- Verifying data constraints and ranges
- Implementing business rule validation
- Enforcing required field checks
- Creating reusable validation utilities
- Providing validation error messages
- Validating file uploads
- Implementing conditional validation
- Validating data before storage or processing
